Etiquetas 2D

Soportado el etiquetado 2D de polígonos o puntos en feature layers y CSV layers. Estos etiquetados se implementan en cliente mediante WebGL.

Mapa de Calor

Soportado los renderizados mediante mapas de calor. Puede ser definido mediante código, o creado en ArcGIS Online/Portal guardado en webmaps.

Mejoras en la parte cliente

Añadir feature layers con cientos de miles de puntos y mantener un alto rendimiento. Los resultados se procesan de forma dinámica a medida que navegamos por el visor.

Color de Fondo personalizado

Posibilidad de embeber escenas web con un fondo personalizado mediante el cambio de color para que se asemeje más al estilo del sitio destino.

Mejoras bajo tierra

Visualizar escenas bajo tierra (escenas locales y gloabales). El suelo puede establecerse parcialmente transparente y modificarse su color.

Mejoras en Dibujo

El escalado y la rotación están ahora soportados para el sketching y el dibujado. (Nuevo eventos implementados)

Consultas en cliente

Soportadas las consultas espaciales en cliente, consultas mediante buffers de distancia y la capacidad de usar expresiones.

Widget Bookmark

Nuevo widget para mostrar todos los marcadores de nuestros mapas web.

Soporte completo Web Maps

Prácticamente soportados todos los web maps creados mediante el último constructor online.
